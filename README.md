## 实时公交查询

[地址](http://bus.chanjiji.xyz)

背景介绍：穷逼学生出行首选公交，可是被某APP彻底恶心到了，打开和响应速度又慢，广告又多，占用内存又大

项目介绍：'querybus‘初衷是给大家提供一个打开速度更快的，没有广告的公交实时查询Web工具，当然首要就是适配移动端

使用说明：由于作者在广州，暂时只支持广州地区的查询，前端做的有点简陋，欢迎各位提意见
